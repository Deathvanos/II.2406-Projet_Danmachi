Présentation du jeu Danmachi RP

DanmachiRP.fr est un jeu en ligne gratuit dans lequel le joueur incarne un héro afin d’aider sa guilde à prosperer. L'objectif premier est de devenir le meilleur Hero. Pour cela divers moyens existent comme améliorer ses caractéristiques, acheter les meilleurs équipements, atteindre les plus hait sommets du donjon, ... Mais le joueur peut choisir d'autres objectifs selon ses envies : son objectif peut être de collectionner le plus d’items rare possible, de devenir le plus riche héro ou être le meilleur chasseur de monstres du pays.
