Game Master

Qu'est-ce qu'un maître du jeu ?

Il s'agit d'une personne qui joue le rôle des monstres que vous pouvez croiser dans la nature ou dans le donjon. Bien que plus souvent il s'agit des monstres dans le donjon. Il a aussi pour rôle de décrire l'ensemble des choses qui se passe lorsque vous êtes dans le donjon, on l'appelle aussi MJ.

Comment devenir MJ et pourquoi ?

Comment ? Il suffit de passer une pseudo formation qui vérifie que vous connaissez l'ensemble des systèmes et des règles du donjon qui seront écrite ci-dessous. Pourquoi ? Cela vous permet de faire vos donjons à n'importe quel moment, de comprendre comment fonctionne le donjon et de vérifier que personne ne triche. Cependant, vous pouvez également faire sans, une simple vérification des staffs sera alors à demander pour voir si le donjon a été réalisé correctement.

Les étapes préliminaires lorsque vous commencer à faire vôtre travaille de MJ

Avant de commencer à décrire tout ce qu'il va se passer, faire apparaître des monstres il faut faire un travail préliminaire. Noter sur une feuille ou directement ici plusieurs informations, les statistiques de tous les membres, leurs vies et leurs énergies. Vous devez aussi prendre en compte leurs arme et armure. Puis faite votre premier calcul qui sera essentiel, faite la somme des statistiques de tous les aventuriers présents dans le groupe et noter le quelque part. Ce sont les choses à faire avant quoi que ce soit.

L'apparition des monstres

Vous venez de finir vôtre travaille préliminaire maintenant vous voulez faire apparaître des monstres. Observer dans quel étage vous êtes puis aller dans le bestiaire. Prenez en note les monstres qui peuvent apparaître dans l'étage ainsi que leurs statistiques avant de retourner dans le chanel où se trouve les aventuriers. Une fois cela, vous devez savoir qu'il n'existe pas une infinité de monstre et que vous serez donc limiter. Le temps de respawn des monstres est de 1 à 2 heures.

Le combat

Dans cette partie vous aller apprendre comment gérer que ce soit le player vs player (pvp) ainsi que le player vs entite (pve). Les aventuriers sont confrontés face à un obstacle vivant, un combat s'engage alors en tour par tour il faut donc déterminer l'ordre du combat. Pour cela faite une liste de l'entité la plus agile présente, puis la seconde, puis la troisième ainsi de suite. Dans le cas ou deux entités on la même agilité ils jouent en même temps. Surtout n'oublier pas de prendre en compte les concepts de combat.

Le combat se termine quand l'un des deux n'est plus en état de se battre.

Le Loot

Les aventuriers doivent être récompenser d'avoir tué le mal, il y a deux choses que tout monstres peuvent loot, les pierres et l'xp.

  • Les pierres : Vous obtenez une pierre violette à chaque monstres mort.
  • L'xp : Pour compter l'xp gagner par les aventuriers il faut utiliser une formule :

    ({K} x [ Nombre de Monstre ] ) / [Somme des stats des aventuriers ]

    Avec K = une constante prédéterminer pour chaque étage que vous pouvez retrouver dans le bestiaire.
    Note : L'xp doit être calculé après chaque combat et vous devez arrondir chaque résultat.
  • Le loot spéciale : Certains monstres peuvent offrir autre chose que des pierres ou de l'xp. Pour savoir lesquels vous pouvait aller voir dans le bestiaire vous y trouverez les loot qu'il peuvent offrir ainsi que la chance de les obtenir. Pour chaque 1 obtenue cela fait apparaître le loot.
    Dans le même principe que les pierres vous utiliser la commande :

    !roll [nombre_de_monstre_qui_loot]d[chance de loot]

Attention ! Après chaque combat le loot doit être écrit dans le channel dans lequel les aventuriers se trouve. De plus après chaque donjon vous devez notée l'xp gagner par les aventuriers dans xp points de la manière suivante :

Nom : {XP avant le combat} + {XP gagné } = {XP Total} Energie [énergie actuelle]/[énergie max] {Hp actuelle}/ {Hp max}
